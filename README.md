## Installation

1. Clone repo

```bash
$ git clone https://github.com/mcruise40/nedi-dockerized.git
```

2. Run docker compose

```bash
docker compose up -d
```

## Backup

Save the Docker volumes **mysqldata** and **nedifiles**.
